|---|--------------------------------------------|-----------------------------------------------------------------------------------|
| 1 | [Selection Sort](./SortingAlgorithms.java) | Repeatedly selects minimum element. Time: **O(n¬≤)**. Not stable.                  |
| 2 | [Bubble Sort](./SortingAlgorithms.java)    | Repeatedly swaps adjacent elements. Time: **O(n¬≤)**. Stable.                      |
